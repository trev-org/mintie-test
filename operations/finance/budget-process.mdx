Intuit's fiscal year runs **August through July**. The budget cycle:

| Time           | Activity                                                  |
| -------------- | --------------------------------------------------------- |
| Feb–Mar        | Strategic planning kickoff (3-year LRP)                    |
| Apr–May        | Bottoms-up budget proposals from BUs                       |
| Jun            | Trade-offs, calibration, board prep                        |
| Jul            | Board approval, budget locked                              |
| Aug            | New FY begins                                             |
| Quarterly      | Forecast updates (Re-forecast 1, 2, 3, 4)                  |

## Cost types

| Type             | How it's tracked                                  |
| ---------------- | ------------------------------------------------- |
| Headcount        | Per-person, FT/contractor, fully-loaded            |
| OpEx             | Expense by category and cost center                |
| CapEx            | Capitalized software, hardware, real estate        |
| Vendor / Cloud   | Tracked via [Procurement](/operations/finance/procurement) |
| Marketing spend  | Separate planning track, paid media-heavy          |

## Cost centers

Every team has a 4-digit cost center. Examples:

- 4501 — QBO Engineering
- 4502 — Payroll Engineering
- 4801 — TurboTax Engineering
- 5101 — Genos AI Platform
- 7001 — DEI Office
- 9001 — IT (cross-charged)

Charge codes appear on POs, expense reports, and Workday transactions.

## Forecasting

Each BU re-forecasts at the start of every quarter:

- Variance vs. plan (last quarter actuals)
- Updated forecast for remaining FY
- Risks and opportunities
- Headcount status

VPs review with CFO; CFO consolidates for the board.

## Hiring approvals

Hiring is gated by budget:

- New roles: must be in approved plan, OR re-allocated from another open req
- Backfills: typically auto-approved if within the planned headcount
- Mid-year additions: VP + CFO approval

## What "spending efficiently" means here

The CFO's question to every team is: "If you got 10% more budget, what would you do? What if you got 10% less?" — coming up with answers is part of the planning process.
