The Software Catalog is the place to request any software, SaaS, or internal tool. It's at `software.intuit.example` (Okta SSO).

## Baseline software

Everyone gets these on day one — no request needed:

- Google Workspace (Gmail, Calendar, Drive, Meet)
- Slack
- Zoom
- Chrome / Edge / Firefox / Safari
- Office 365 (most employees)
- 1Password
- VS Code (engineers)
- IntelliJ Ultimate (engineers)
- Cursor (with manager approval)

## How to request

<Steps>
  <Step title="Search the catalog">
    Type the software name. Already on the list? You'll see access requirements and a request button.
  </Step>
  <Step title="Justify">
    Brief reason. Be specific — "for migrating customer data" beats "for work."
  </Step>
  <Step title="Manager approval">
    Auto-routed to your manager. Typical turnaround: same day to 1 business day.
  </Step>
  <Step title="Provisioning">
    On approval, IT provisions and emails you sign-in instructions.
  </Step>
</Steps>

## What requires extra review

| Trigger                                 | Additional review                    |
| --------------------------------------- | ------------------------------------ |
| Software not in catalog                 | Vendor due diligence (4-6 weeks)     |
| Tool that handles customer data         | Privacy + Security review            |
| Tool replacing baseline software        | IT architecture review               |
| Annual cost > $5,000                    | Finance approval                     |
| Open-source library for production use  | License review, SCA scan             |

## Adding a new vendor

If a tool isn't in the catalog, propose it via the **New Vendor Request** form. The intake includes:

- Vendor name and product
- Use case and business justification
- Estimated annual spend
- Number of users
- Will customer data flow to this vendor?
- Do you have a contract draft?

The Vendor Management Office (VMO) coordinates Privacy, Security, Legal, Finance review. Plan **6 weeks** for a new vendor with customer-data implications; **1-2 weeks** for safe non-data tools.

## License audit

Annual audit reconciles entitlements with usage. Inactive licenses (90+ days no use) are reclaimed automatically. You can re-request if needed.
