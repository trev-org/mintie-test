Print this, screenshot it, or check items off in Workday — whatever helps. The goal: by Friday of week one, you have access to everything you need and have shipped at least one thing.

## Monday

- [ ] Laptop powered on, Okta MFA enrolled
- [ ] Slack profile completed (photo, pronouns, role, location)
- [ ] Email signature updated using the [official template](https://brand.intuit.example/email-signature)
- [ ] Calendar set with working hours and holidays
- [ ] Joined `#new-hires` and intro'd yourself
- [ ] 1:1 with manager scheduled (recurring, 30 min, weekly)

## Tuesday

- [ ] Health, dental, vision, and 401(k) elections started in Workday — see [Benefits overview](/people/benefits-overview)
- [ ] I-9 verification submitted (U.S. only) — must be complete by **end of day 3**
- [ ] Direct deposit configured in Workday
- [ ] Read your team's onboarding doc end to end
- [ ] Met your onboarding buddy

## Wednesday

- [ ] Engineers: dev environment running locally — see [Dev environment](/engineering/dev-environment)
- [ ] Engineers: GitHub Enterprise access confirmed, you can clone a service repo
- [ ] Joined product- and team-specific Slack channels
- [ ] Skimmed the [Service catalog](/engineering/services/service-catalog) for your domain

## Thursday

- [ ] Required training started: Code of Conduct, Security Awareness, Privacy Basics
- [ ] Skip-level 1:1 scheduled (within first 30 days)
- [ ] Reviewed [Operating values](/culture/operating-values) and discussed with manager
- [ ] (Engineers) Shipped first PR — even if it's a typo fix in a README

## Friday

- [ ] Retro with your manager: what's clear, what's still murky
- [ ] Filled out the new hire survey (link in `#new-hires`)
- [ ] Goals drafted in Workday for next 30 days
- [ ] Confirmed you can access at least three internal tools without help: Okta, Workday, GitHub
