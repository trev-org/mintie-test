QuickBooks is the largest SBSE product family. It started as desktop in the 1990s, became cloud-first in the 2010s, and is now an integrated platform spanning accounting, payroll, payments, time, capital, and more.

## Multi-tenancy

QBO is multi-tenant at the **realm** level. A realm is one company file (small biz). Users can belong to multiple realms (e.g., an accountant with 50 clients).

Tenant isolation:

- Per-realm database schema in shared Aurora clusters (sharded)
- Realm-scoped IAM checks at every API
- Per-realm encryption keys (KMS-derived) for sensitive fields

## Scale snapshot (FY25)

- ~9M active QBO subscribers globally
- ~250M API calls per day
- ~12B accounting transactions per month
- ~1.5M payroll runs per quarter
- ~$200B in invoices created annually
- ~$70B in payments processed annually

## What internal teams should know

- QBO is **always on**. There is no off-season — every business needs accounting every day.
- Multi-currency, multi-language, multi-tax-jurisdiction. International is huge.
- Accountants are a critical user segment. The "ProAdvisor" experience is a separate UX surface.
- Banking feeds break — it's the most-customer-facing reliability concern.
