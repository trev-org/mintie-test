# **MM8108-MF15457 ハードウェア設計ガイド**

MM8108-MF15457 は、[Wi-Fi HaLow / IEEE 802.11ah 規格](https://en.wikipedia.org/wiki/IEEE_802.11ah) を用いた長距離・低消費電力の無線通信向けに設計された Sub-GHz トランシーバの System-on-Module (SoM) です。RF ベースバンド、媒体アクセス制御 (MAC)、デジタルインターフェースを、組み込みおよび IoT 用途に最適化されたコンパクトで低消費電力なデバイスに統合しています。

MM8108-MF15457 は SDIO または SPI 経由でホストアプリケーション MCU と統合でき、アプリケーション固有のカスタムピンをサポートする GPIO も提供されます。モジュールのプログラミングと診断のための JTAG (Joint Test Action Group) ピンも用意されています。
