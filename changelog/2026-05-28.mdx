## New

- **MM6108 SoC datasheet.** Full specification for the MM6108 Wi-Fi HaLow (IEEE 802.11ah) single-chip SoC is now published, including radio, PHY, MAC, power, and interface details. Use it to design and integrate the MM6108 into sub-1 GHz IoT products. See the [MM6108 Data Sheet](/MM6108_Data_Sheet).
- **MM8108-MF15457 hardware design guide.** New design guide for the MM8108-MF15457 module covering schematic, layout, and integration guidance. See the [Hardware Design Guide](/MM8108-MF15457_Hardware_Design_Guide_0_1_1).
- **HaLowLink user guide.** Setup and operation guide for the HaLowLink gateway. See the [HaLowLink User Guide](/HaLowLink_User_Guide_0_1_4).
- **EKH05 Evaluation Kit user guide.** Getting started, board overview, and usage details for the EKH05 evaluation kit. See the [EKH05 User Guide](/EKH05_Evaluation_Kit_User_Guide_0_1_4).
- **Application Note 38.** New guide on using Wi-Fi EasyConnect (DPP) with the HaLowLink for device provisioning. See [App Note 38](/AppNote_38_version_0_1_3).

- **BCF Editor v3.4.1.** New BCF Editor release available for download.
- **BCF Editor v2.9.1.** Maintenance release of the v2.x BCF Editor remains available alongside v3.x.
