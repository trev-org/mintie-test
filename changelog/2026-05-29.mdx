## New

- **MM8108-MF15457 module datasheet.** Full technical specification for the MM8108-MF15457 multi-region Wi-Fi HaLow (IEEE 802.11ah) module is now published. Covers the sub-1 GHz radio, integrated 26 dBm PA, USB 2.0 and SDIO/SPI host interfaces, and PHY rates up to 43.3 Mbps. Use it to design and integrate the module into IoT products. See the [MM8108-MF15457 Data Sheet](/MM8108-MF15457_Data_Sheet).
