## New

- **Application Note 43 — HaLow RF Test Guide.** Detailed guidance for conducting and interpreting cabled RF measurements of MM6108 and MM8108 Wi-Fi HaLow devices. Covers RF measurement parameters, HaLow test modes, VST-based TX and RX evaluation, test environment setup with LitePoint testers, and use of the Morse Micro RF Tester software. Use it to configure and run consistent, repeatable RF measurements on HaLow devices. See [App Note 43](/AppNote_43_RF_Test_Guide).
