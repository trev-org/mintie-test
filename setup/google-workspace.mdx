## Setting up today (manual path)

Use the [generic SAML / OIDC](/setup/generic-saml) flow with Google's standard OIDC endpoints:

| Mintie field | Value |
| --- | --- |
| Issuer | `https://accounts.google.com` |
| Authorize URL | `https://accounts.google.com/o/oauth2/v2/auth` |
| Token URL | `https://oauth2.googleapis.com/token` |
| JWKS URL | `https://www.googleapis.com/oauth2/v3/certs` |
| Client ID / Secret | From Google Cloud Console → APIs & Services → Credentials |

<Tip>
  The dedicated connector unlocks Directory Sync. Without it, Google
  Workspace works for SSO but not for auto-provisioning — new hires need to
  be invited manually until SCIM or the connector lands.
</Tip>
