# HaLow 射频测试指南

<Info>
  APPNOTE-43 · 射频测试指南 v1 · 专有及保密
</Info>

## 范围

本应用笔记提供详细的技术指南与基础背景知识,用于对 MM6108 与 MM8108 Wi-Fi HaLow 设备进行有线射频 (RF) 测量并解读测量结果。

本文档介绍基础理论概念,包括关键的射频测量参数、HaLow 测试模式,以及使用矢量信号收发器 (VST) 进行发射 (TX) 和接收 (RX) 评估的方法。文档还全面概述了测试环境的搭建,详细说明 LitePoint 测试仪、被测 HaLow 设备 (DUT) 及相关射频仪器等关键组件,并提供使用 Morse Micro 提供的 RF Tester 软件包配置和控制被测 HaLow 设备的指导。
