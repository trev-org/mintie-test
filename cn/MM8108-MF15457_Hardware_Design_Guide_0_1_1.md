# **MM8108-MF15457 硬件设计指南**

MM8108-MF15457 是一款 Sub-GHz 收发器系统级模组 (SoM),基于 [Wi-Fi HaLow / IEEE 802.11ah 标准](https://en.wikipedia.org/wiki/IEEE_802.11ah) 提供远距离、低功耗的无线通信。它将 RF 基带、介质访问控制 (MAC) 与数字接口集成于一颗紧凑、低功耗的器件中,针对嵌入式与物联网 (IoT) 应用进行了优化。

MM8108-MF15457 可通过 SDIO 或 SPI 与主机应用 MCU 集成,并提供 GPIO 以支持自定义的、特定于应用的引脚。模组还提供 JTAG(联合测试行动小组)引脚,用于模组编程和诊断。
